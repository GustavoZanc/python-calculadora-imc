# Calculadora de IMC

Projeto simples em Python para calcular o IMC de uma pessoa e classificar o resultado.

## O que pratiquei

- Entrada de dados
- Funcoes
- Condicionais
- Formatacao de texto

## Como executar

```bash
python imc.py
```

## Exemplo

```text
Peso: 72
Altura: 1.75
IMC: 23.51
Classificacao: Peso normal
```
